# Pragmatis_Analytics

Olá, tudo bem, caso queira rodar esse aplicativo em seu computador, será necessário instalar os pacotes presentes em 'requirements.txt', para isso basta fazer  pip install -r requirements.txt no seu enviroment. O arquivo 'decolar.py' realiza o web scraping via código diretamente e nos retorna um output com a busca desejada para isso é necessário o arquivo de input: 'input.xlsx' no mesmo diretório que o código esteja.

O arquivo 'main.py' é a aplicação por meio da lib streamlit, para executá-lo basta ter os pacotes do 'requirements.txt' no sistema e executá-lo. Em seguida será aberto um localhost no qual irá rodar o app. O arquivo 'setup.py' é uma tentativa de criar um executável, porém o streamlit não é eexecutado por esse comando.
